# Web Development Experiments

This repository contains 10 web development experiments as part of the curriculum.

## List of Experiments

1.  **exp1:** HTML5 Form and Media Elements.
2.  **exp2:** HTML5 and Advanced CSS (Flexbox, Grid, Animations).
3.  **exp3:** Responsive Web page using Bootstrap 5.
4.  **exp4:** Javascript DOM elements and Events (Task List Manager).
5.  **exp5:** Javascript Time Functions and Objects (Digital Clock & Theme Switcher).
6.  **exp6:** jQuery (Accordion & Effects).
7.  **exp7:** JSON and NoSQL concepts (Data retrieval simulation).
8.  **exp8:** Node.js with a Database module (Express & SQLite).
9.  **exp9:** NoSQL Database - MongoDB (Express & Mongoose).
10. **exp10:** AI Tools (Web Speech API AI Voice Assistant).

## How to Run

- For **exp1-exp6** and **exp10**, simply open the `index.html` file in any modern web browser.
- For **exp7**, run `node app.js` in the `exp7` directory.
- For **exp8**, run `npm install` and then `npm start` in the `exp8` directory.
- For **exp9**, run `npm install` and then `npm start` in the `exp9` directory (requires MongoDB).
